# mailman3-install

Description
===========
Scripts to easily create a functional Mailman Suite 3.x installation.




Requirements
============
Tested with

* Mailman 3.2
* Postfix 3.4
* Certbot 0.40
* Ngnix 1.18
* Ubuntu 20.04
* ...


Usage
=====

Download:

	$ git clone https://github.com/jdkelleher/mailman3-install.git

* Do the stuff ...
